¿Qué ves aquí?
  • Comparación directa por dataset: modelos base vs su versión Local.
  • Barras de accuracy y diferencial (Local vs base) por clasificador.
Cómo usarla
  • Elige iteraciones (10/100), discretizador base (mdlp, igual freq/amp, PKI) y puntos de corte.
  • Cambia de dataset con los controles de navegación de la tarjeta.
Qué preguntas responde
  • ¿Cuánto mejora o empeora Local frente al mejor base en este dataset?
  • ¿Cómo rinde cada clasificador (TAN, KDB, AODE) con el mismo discretizador/corte?
Consejos
  • Usa 10it para las conclusiones principales y recorre datasets para ver patrones.
  • Cambia discretizador base para ver si las mejoras dependen del corte.
